# EV_Charging_Time_Tool

This project analyzes electric vehicle charging data using Python to study charging behaviour and energy usage.

## Project Overview
The tool processes EV charging data to understand charging patterns, charging time, and power consumption during charging cycles.

## Features
- Charging power analysis
- Charging time monitoring
- Energy consumption visualization
- Charging efficiency insights
- Graphical representation of charging behaviour

## Technologies Used
- Python
- Pandas
- Matplotlib
- Jupyter Notebook

## Output
The project produces graphs that help visualize EV charging performance and energy usage patterns.

## Applications
EV charging analysis helps understand charging efficiency and optimize charging infrastructure for electric vehicles.
